### Kết quả tuần 4:
**Tổng quan:**

Ở tuần 4, tôi tập trung vào Amazon EC2 như một máy chủ ảo trên AWS. Phần thực hành giúp tôi quen hơn với việc tạo instance, cấu hình Security Group, kết nối máy chủ và chạy một web server cơ bản.

**Kiến thức lý thuyết tiếp thu:**

* Amazon EC2.
* AMI, Instance Type và Key Pair.
* Security Group cho EC2.
* Linux Instance và Windows Instance.
* EBS Volume.
* Cách kết nối SSH, EC2 Instance Connect và RDP.

**Thực hành với bài lab:**

* Tạo EC2 Instance.
* Kết nối Windows Instance bằng RDP.
* Kết nối Linux Instance bằng SSH hoặc EC2 Instance Connect.
* Cài đặt web server cơ bản.
* Kiểm tra truy cập ứng dụng qua public IP.
* Dọn dẹp tài nguyên EC2 sau khi hoàn tất lab.
