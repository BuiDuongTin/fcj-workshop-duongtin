### Week 1 results:
**Overview:**

This week helped me get used to the AWS learning environment and understand the basic tools for tracking cost, getting support, and working with services.

**Knowledge learned:**

* AWS Free Tier is useful for early practice, but cost tracking is still necessary.
* AWS Budget helps monitor spending and send alerts when costs approach a threshold.
* AWS Support provides help when technical issues occur.
* AWS CLI makes AWS tasks faster and more convenient than manual Console work.
