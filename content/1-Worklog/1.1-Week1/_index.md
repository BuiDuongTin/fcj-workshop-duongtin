### Week 1 results:
**Overview:**

The first week helped me settle into the program and understand how AWS fits into cloud learning. I became more comfortable with the Console and CLI, and I started to recognize the basic layers of an AWS architecture.

**What I learned:**

* AWS is a common platform for modern cloud systems.
* Free Tier is useful for practice, but costs still need attention.
* The Console is better for visual management, while the CLI is useful for repeatable work.
* A basic AWS diagram should show network, compute, and storage clearly.

**Hands-on practice:**

* Signing in and exploring the AWS account.
* Installing AWS CLI and checking its configuration.
* Browsing services in the Console.
* Sketching a basic AWS architecture in my own words.
